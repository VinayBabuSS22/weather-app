## 📖 About the Project

This is a personal learning project created during my **second year of studies** to practice and refine front-end development skills. It focuses on modern responsive layouts, handling asynchronous HTTP requests, DOM manipulation, and dynamic media loading. 

The application fetches and displays real-time weather information and 5-day forecasts for any city around the globe.

---

## 🚀 Key Features

*   **Real-time Weather Data**: Instantly fetch current weather conditions for any searched city.
*   **Dynamic Background Scenes**: The application background changes dynamically based on the current weather condition (Clear, Cloudy, Rainy) and the time of day (Day or Night) to create an immersive user experience.
*   **Comprehensive Weather Details**:
    *   Current Temperature
    *   "Real Feel" (apparent temperature)
    *   Maximum & Minimum daily temperatures
    *   Humidity levels
    *   Barometric Pressure
    *   Visibility range
    *   Wind speed & precise compass directions (e.g., `NE`, `SW`)
    *   Sunrise and Sunset times formatted in local time
*   **5-Day Daily Forecast**: Displays a scrollable grid of forecast cards for upcoming days with daily temperature ranges and weather icons.
*   **Unit Toggle**: Easily switch between Metric (°C, km/h) and Imperial (°F, mph) systems.
*   **Smart Prefetching**: Prefetches high-quality background images to ensure smooth transitions when changing backgrounds.
*   **Responsive Layout**: Fully optimized for mobile and desktop screens, featuring a custom mobile search bar and sliding views.

---

## 🛠️ Tech Stack & Integrations

*   **Structure**: HTML5
*   **Styling**: Vanilla CSS3 (Custom fonts, HSL color palette, Glassmorphism design elements, CSS media queries)
*   **Logic**: Asynchronous JavaScript (ES6+ `async/await` and Fetch API)
*   **API**: [OpenWeatherMap API](https://openweathermap.org/) (Current Weather & 5-Day / 3-Hour Forecast feeds)

## 💻 How to Run Locally

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Vinaysetty22/weather-app.git
    ```
2.  **Navigate to the project directory**:
    ```bash
    cd weather-app
    ```
3.  **Open in your browser**:
    Simply open the `index.html` file in any modern web browser, or run it using a local development server like VS Code's "Live Server" extension.
